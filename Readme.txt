Welcome to Regnow Site Builder! This is a maintenance release of the
Template of Regnow Site Buider.

System Requirements
---------------------------------------
* web server: Apache2 or above
* php compiler: php5.0 or above

How to Use?
---------------------------------------
* Unpack the downloaded zip file 
* If network server does not support "allow_url_fopen", please edit includes\configuration.ini by setting remote_aceess=curl.
* Relocate these contents to web server

Directories
---------------------------------------
images			--image directory of webiste
js			--javascript directory
styles			--css directory
includes		--php library files
  |-system.php		--site config
  |-configuration.ini --Configure File
  |-configuration_funs.php --Site Configuration File Function
  |-rootdir		--get the main directories of website
  |-dao			--get RegNow data
    |-cache		--data cache directory
  |-ui			--SEO friendly URL & paging controller
  |-widget		--widget library
    |-base		--Control on widgets' basics
    |-html		--Control on widgets' display
